  • debounce(fn, time?, immediate?): ((this: any, ...args: any[]) => void)

  • 实现函数防抖 防抖是指在事件被触发后，不会立刻执行函数，而是等待指定的时间间隔 如果在这段时间内事件再次被触发，则重新计时，只有当这段时间内没有新的触发时，函数才会执行 这可以用于优化性能，减少因频繁操作而引起的不必要的计算或渲染

    Parameters

    • fn: Function

      要防抖的函数

    • time: number = 1000

      等待的时间间隔，以毫秒为单位，默认为1000毫秒

    • immediate: boolean = false

      是否在第一次触发时立即执行函数，默认为false

  • isType<T>(type, obj): boolean

  • 判断给定对象是否为指定的类型

    此静态方法通过比较对象的内部类型与指定类型是否匹配来判断 它使用了Object.prototype.toString.call(obj)方法来获取对象的内部类型，然后通过比较其切片后的字符串（去除了前后缀）与指定类型的字符串是否一致来判断类型是否匹配
